# AI-Tool-Dashboard-Power-Bi
AI-Tool-Dashboard is a Power BI project that analyzes AI tool usage, effectiveness, and productivity impact across departments. It includes an interactive dashboard (.pbix file), dataset, and screenshots for quick preview

## 📊 Overview

This project is a Power BI dashboard analyzing AI tool usage, effectiveness, and productivity impact across departments.

## 🎯 Key Features

* KPI tracking (Usage Hours, Effectiveness, Productivity Gain)
* Department-wise AI usage analysis
* Tool comparison (ChatGPT, Gemini, Copilot)
* Trend analysis over time

## 🛠️ Tools Used

* Microsoft Power BI
* DAX (basic measures)
* Data Visualization

## 💡 Key Insights

* IT department shows highest AI adoption
* ChatGPT has highest effectiveness
* Productivity increased by 25% with AI usage
